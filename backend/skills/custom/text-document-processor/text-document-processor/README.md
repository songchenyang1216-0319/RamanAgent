# Text Document Processor Skill

This skill is used for processing text documents.

## Main features

- Translation
- Bilingual comparison
- Summary
- Polishing
- Rewriting
- Markdown formatting
- Key point extraction
- Study note generation
- Speech draft generation

## Required file

- `SKILL.md`

## Optional folders

- `references/`: detailed rules
- `assets/`: templates
- `scripts/`: optional executable tools
